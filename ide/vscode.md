# Vscode

ctrl+shift+p

## EntryPoint

- > open user settings 用户设置
- > file icon theme 文件图标
- > keyboard shortcuts 快捷键
- > color theme
- > run task

## Editor

- `alt+shift+多行` 多行同时输入

    ```text
    alt+shift+选中多行， 开始输入
    ```

- `> fold all` 折叠
- `> unfold all`
