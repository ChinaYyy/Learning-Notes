# Package Directory


```plain
├─ src
│  └─ packagename
│     ├─ __init__.py
│     └─ ...
├─ tests
│  └─ ...
└─ setup.py
```

test也可以放在代码内部

```plain
├─ src
│  └─ packagename
│     ├─ __init__.py
|     ├─ ...
|     └─ tests
|        └─ ...
└─ setup.py
```
