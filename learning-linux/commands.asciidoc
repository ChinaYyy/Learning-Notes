# Commands

## ls
- -a --all包括隐藏文件
- -l 详细信息(long)
- -h size数据人类可读
- -i inode值
- -d 目录信息

----
-rw-r-----
----
首字符 `-` 表示文件； `d` 表示目录； `l` 表示软链接

分别对应 `user`,`group`,`other` 的权限

## mkdir

- -p --parents 递归创建

## cp

----
cp [option] file1 file2 desddir
----

- -r 对目录下面的内容递归操作
- -p --preserve 保留指定的属性（模式，所有权，时间戳）

NOTE: 可以操作多个路径，也可以改名

## 文件操作

- touch
- cat [-n] 显示行号
- more/less
- head [-n]
- tail [-nf]

## ln

----
ln -s /etc/services /tmp/services.soft
ln /etc/services /tmp/services.hard
----

## chmod

针对文件

- `r 4 cat/more/less/head/tail`
- `w 2 vim`
- `x 1 script command`

针对目录

- `r 4 ls`
- `w 2 touch/mkdir/rm`
- `x 1 cd`

----
chmod 777 somefile
chmod 777 somedir
chmod -R 777 somedir
----

`[-R]` 递归改变子文件夹及子文件的权限


## chown/chgrp

----
chown yyy somedir
chgrp [group] somedir
----

NOTE: 只有管理员才有 `chown` 的权限


## find

1. 根据文件名称搜索
+
----
find /etc -name init     //精确搜索
find /etc -name *init*   //通配符
find /etc -iname init    //忽略大小写
----

2. 根据文件大小搜索 +
size需要换算成数据块：1数据块=0.5k +
100M = 100*1024＊2 = 204800
+
----
find -size +204800   //大于100M
find -size -204800   //小于100M
----

3. 根据用户查找
+
----
find /home -user yyy
find /home -group yyy
----

4. 根据时间属性查找
+
----
find /etc -amin -5    //访问时间access 5分钟
find /etc -cmin -5    //文件属性change 5分钟
find /etc -mmin -5    //文件内容modify 5分钟
----

5. 根据文件类型查找
+
----
find /etc -type f    // f文件
find /etc -type d    // d目录
find /etc -type l    // l软链接
----

6. 逻辑查找
+
`-a` 表示and，`-o` 表示or
+
----
//查找大于80M小于100M的文件
find /etc -size +163840 -a -size -204800
----

## grep

文件内容查找 +
`-i` 不区分大小写 +
`-v` 排除指定字符串
----
grep mysql /root/install.log
grep -v ^# /etc/inittab        //删除#号开头的行
----

----
